# CK-Map

## Map Capture Window

Always have a map visible. 

## Installation

To install CK-Map, have the CK package installed and run

`CK pkg install map`

`CK pkg update map`

## Usage

Once installed a window will popup containing the last displayed map

### Aliases

* `CK map show`
  * If you close the Map window this will show it
* `CK map hide`
  * hide the map
